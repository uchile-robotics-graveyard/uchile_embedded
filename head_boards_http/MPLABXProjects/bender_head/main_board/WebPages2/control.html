Control commands

Send control commands using the HTTP's GET method to the control.html form. e.g: http://~config_ip~/control.html?emotion=serious&yaw=0&mouth=off"

Key-value constraints:

  • emotion ∈ {serious, happy, happy1, happy2, happy3, sad, sad1, sad2, sad3, angry, angry1, angry2, angry3, surprise, surprised, shame, ashamed, standby, eyebrow, greetings, 1313, ear, yes, no, lost, relaxed, normal, agitated, blink, flirt}
  • yaw ∈ [~yaw_min~ ~yaw_max~] [deg]
  • mouth ∈ {on, off}
  • string values are case insensitive (the same as URLs)
  • 'happy', 'sad' and 'angry' values will be read as happy3, sad3 and angry3.
